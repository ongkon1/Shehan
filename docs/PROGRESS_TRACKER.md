## How to Update

When completing a task:

1. Change `- [ ]` to `- [x]` for the task
2. Update the **Done** count in the Milestone Breakdown table
3. Update the progress bar using: `█` for filled, `░` for empty (10 blocks = 100%)
4. Update the **Overall Progress** section totals
5. Change milestone **Status** as needed:
   - `⬜ Not Started` → `🔶 In Progress` → `✅ Completed`
6. Update the **Last Updated** date at the top

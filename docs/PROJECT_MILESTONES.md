# Portfolio Website — Project Milestones

Build in this order. Complete each milestone fully before moving to the next.

---

## Milestone 1: Project Setup & HTML Structure

**Goal:** Scaffold the project and build the complete HTML skeleton.

| # | Task | Status |
|---|------|--------|
| 1 | Create folder structure (`css/`, `js/`, `assets/`) | ✅ |
| 2 | Create `index.html` with all semantic sections (nav, hero, about, skills, projects, contact, footer) | ✅ |
| 3 | Add meta tags (viewport, description, Open Graph) | ✅ |
| 4 | Link CSS file, JS file, Google Fonts, and Lucide Icons CDN | ✅ |
| 5 | Add placeholder content for all sections | ✅ |

**Done when:** Opening `index.html` in a browser shows all sections with raw unstyled content.

---

## Milestone 2: CSS Styling & Theming

**Goal:** Style every section, implement responsive layout, and add dark/light theme support.

| # | Task | Status |
|---|------|--------|
| 1 | Define CSS custom properties for light and dark theme colors | ✅ |
| 2 | Style the navigation (sticky, responsive) | ✅ |
| 3 | Style hero section with entrance animation (CSS only) | ✅ |
| 4 | Style about section | ✅ |
| 5 | Build responsive grid for skills cards | ✅ |
| 6 | Build responsive grid for project cards with hover effects | ✅ |
| 7 | Style contact form and footer | ✅ |
| 8 | Implement dark/light mode via `[data-theme="dark"]` on `<html>` | ✅ |
| 9 | Add `prefers-color-scheme` media query for default theme | ✅ |
| 10 | Add responsive breakpoints (768px, 480px) | ✅ |
| 11 | Style buttons with hover/focus states | ✅ |

**Done when:** The site looks polished in both light and dark mode across desktop, tablet, and mobile.
